Description
combine multiple images into 1 and choose png or jpeg as the output

URL
 http://xaymacastudios.appspot.com/image/composition


Available Formats
xml

HTTP Method
PUT

Requires Authentication
False

Headers
"Content-Type: text/xml"

Parameters
xml file describing how 1 or more images should be composited

example xml:


<?xml version="1.0" encoding="UTF-8"?>
<!-- rename to have an .xml extention, guru does not allow "xml" attachment -->
<compose>
<composite imgKey="049b2d23-4d65-48dd-9c00-6453fc77bb4e" xOffset="10" yOffset="0"  opacity="0.5" anchor="BOTTOM_RIGHT" />
<composite imgKey="69ad8c3c-73bb-4203-9ebe-6e55803d49ae" xOffset="5" yOffset="0"  opacity="0.5" anchor="CENTER_CENTER" />
<finalImage width="300" height="200" color="0xffffffff" encodeAs="PNG" /> 
</compose>







Responses
If you receive a HTTP 200 OK header, you can not assume the image was successfully uploaded. Check the key field for null.
Will change status code on the next version.

Sample  Response

    <?xml version="1.0" encoding="UTF-8"?>
    <image>
    <key>562a50c2-d238-4833-bad3-e26c537d60c3</key>
    <fileURL>/image/562a50c2-d238-4833-bad3-e26c537d60c3/</fileURL> 
    </image>

	

	
	
Example CURL Command
 
curl -v -X PUT  -d @livecompose.xml --header "Content-Type: text/xml" http://xaymacastudios.appspot.com/image/composition
